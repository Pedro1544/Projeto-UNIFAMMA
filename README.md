# Visto.io - Projeto de Extensão Unifamma

Este repositório contém o código-fonte desenvolvido para o **Projeto de Extensão** da faculdade **Unifamma**.
O objetivo deste projeto foi o de desenvolver um sistema WEB no formato de "landing page" (Página de Aterrissagem) para uma empresa, com a finalidade de atuar como um sistema de divulgação de produto/serviço.

## 🏢 Sobre a Empresa Fictícia (Visto.io)
A Visto.io é uma agência de intercâmbio inteligente que utiliza inteligência artificial e consultoria especializada para conectar alunos e profissionais aos melhores destinos acadêmicos e trabalhistas pelo mundo (Canadá, Austrália, Irlanda, etc).

## 🚀 Tecnologias Utilizadas (Front-end)
Este projeto foi desenvolvido focando em performance, zero dependências pesadas e um design premium moderno (Vanilla DOM):

- **HTML5:** Estrutura semântica e SEO otimizado.
- **CSS3 (Vanilla):** Estilos avançados, Grid Layout, Flexbox, esquema de cores "Dark Mode", efeitos de vidro (Glassmorphism) e design totalmente responsivo.
- **JavaScript (Vanilla):** Interatividade de página sem o uso de bibliotecas, incluindo transições suaves de scroll e animações nos elementos usando _Intersection Observer_.

## 🛠️ Como executar o projeto
1. Faça o download ou clone este repositório.
2. Na pasta do projeto, encontre o arquivo principal `project.html`.
3. Dê dois cliques em `project.html` para abri-lo e visualizá-lo em qualquer navegador de internet moderno (Google Chrome, Edge, Firefox, etc.). Não é necessária a instalação de bibliotecas.
